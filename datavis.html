Data Visualization Project




The goal of this project is to create a site to analyze complex cellular segmentation data. Data collected from CellPose in the form of tif and seg.npy files would be able to uploaded and processed using python. An easy method of selecting what you would like to visualize will trigger certain code to run and calculate certain parameters. This would make analyzing data from CellPose much more efficient and allow for researchers to be able to analyze their data and take thier next steps. This program would also allow the user to take a look into individual datapoints to identify errors instead of needing to dive deep into code to identify the error.
